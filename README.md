# Personal Portfolio Website

This is my personal portfolio website, created as a web development assignment. It is a fully responsive, modern website designed to showcase my skills, projects, education, and contact information.

## 🚀 Features
- **5 Complete Pages:** Home, About, Skills, Projects, and Contact.
- **Responsive Design:** Works perfectly on mobile phones, tablets, and desktop screens.
- **Modern UI:** Clean layout with beautiful gradients, rounded cards, and shadow effects.
- **Animations:** Smooth fade-in and pulse animations for a better user experience.
- **Functional Forms:** A clean contact form layout (UI only).

## 🛠️ Technologies Used
- HTML5
- CSS3
- Tailwind CSS (via CDN for styling)
- JavaScript (Basics)
